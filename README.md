App que permite listar, editar, crear y eliminar tareas, listandolas por estado de cada tarea y permitiendo cambiar de estado cada tarea de dos maneras, la primera es arrastrando la tarea de un estado a otro, y la segunda forma es presionando la tarea 
enviando a la pantalla para editar la tarea permitiendo cambiar el estado y tambien te permite editar la tarea. 

En el archivo word se encuentra el requerimiento.
Tambien existe la documentacion de las api con Swagger ingresando a la url /api/docs/
